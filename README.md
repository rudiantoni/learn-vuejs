Alguns projetos não estão usando a CLI do VueJS, portanto foram executados através de um servidor HTTP Apache em um container Docker, para iniciá-los, basta navegar até a pasta do curso `cd course_01` e iniciar a aplicaç~co com `sh [project_name].sh` e acessar através de http://localhost. É importante observar que ele usa a porta 80, então se tiver algo sendo executado nessa porta a aplicação não vai subir. Ao término da utilização, use Ctrl + C para cancelar e remover o container. Os valores de *[project_name]* podem ser:
- [01_iniciando_vuejs_cdn](./course_01/01_iniciando_vuejs_cdn/)
- [02_data_binding](./course_01/02_data_binding/)
